# DeviceThemeProvider
Провайдер обязателен для использования, необходимо обернуть в него все приложение.
Назначение - определяет, на каком девайсе запущено приложение (по умолчанию 3 значения - `sberBox`, `sberPortal`, `touch`)   
В зависимости от значения, применяет глобальные стили типографии, подходящие данной платформе, ко всему приложению, а также расширяет объект темы полем `deviceScale`.  
Тип девайса определяется на основании `userAgent`, при отсутствии `navigator` или `userAgent` девайс  будет определен как `sberBox`.

Имеется возможность передать провайдеру собственную логику определения девайса с помощью пропса `detectDeviceCallback`.   
При этом стоит помнить, что разрешены только 3 стандартных значения.
